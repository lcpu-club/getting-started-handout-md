# 知识速查

虽然笔者本人非常建议同学们通读本手册，但也理解有些同学可能只是想快速查阅知识点或获取某些技能。因此，笔者在此提供一个知识速查表，方便同学们快速定位到相关章。

## 配置相关环境

- 在纯 Windows 上搭建 C/C++ 编程环境：见《在纯Windows上搭建C/C++编程环境》

- 在 WSL 上搭建 C/C++ 编程环境：见《用WSL配置C/C++环境》
