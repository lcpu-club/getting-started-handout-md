# 终端和Linux基础：为开发铺垫语境

## 本章目录

- 终端101
- 开始使用Linux

## 本章摘要

本章介绍命令行界面(CLI)和Linux操作系统的基础知识。首先讲解终端的基本概念、Shell的选择(bash/zsh/fish/PowerShell)以及终端的基本操作,包括命令结构、快捷键、重定向和管道等核心概念。接着介绍Linux系统的基本使用,包括发行版选择、文件系统结构、权限管理、软件安装等内容,以及WSL(Windows Subsystem for Linux)的配置和使用。通过本章学习,读者将掌握使用命令行进行高效操作的能力,为后续的开发工作打下基础。
