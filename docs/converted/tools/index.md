# 工具箱

## 本章目录

- 文字排版：Markdown 和 Typst
- LaTeX
- LaTeX 进阶

## 本章摘要

本章介绍学术写作和文档排版的常用工具。首先讲解轻量级标记语言Markdown的语法和使用,以及新兴的排版系统Typst的基础知识,这些工具适合编写笔记、博客和简单文档。接着系统介绍LaTeX排版系统的基础知识,包括文档结构、数学公式、图表插入、参考文献管理等核心功能,LaTeX是学术论文和技术文档的标准排版工具。最后讲解LaTeX的进阶技巧,包括自定义命令、宏包使用、复杂表格制作、TikZ绘图等高级功能。通过本章学习,读者将掌握专业的文档排版能力。
