# 开发环境：让代码能编译能跑

## 本章目录

- 正式踏入编程世界
- C语言入门
- 从C到C++
- C++进阶
- 现代C++特性
- C++真理大讨论
- C系工程概述
- Python高速入门
- Python常用包

## 本章摘要

本章全面介绍编程开发环境的搭建和编程语言的入门。首先讲解IDE的选择(推荐VS Code)和编译器的安装配置,包括在Windows、Linux和WSL上配置C/C++和Python环境。接着系统讲解C语言的基础语法,包括变量、运算、控制流、数组、指针、文件操作等核心概念。然后介绍C++的面向对象特性、STL标准库、现代C++特性(C++11/14/17/23)以及C++工程实践。最后介绍Python的基础语法、虚拟环境管理(conda/mamba)以及常用的第三方库。通过本章学习,读者将掌握搭建开发环境和编写基础程序的能力。
