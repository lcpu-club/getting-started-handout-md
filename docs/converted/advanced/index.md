# 延伸进阶阅读

## 本章目录

- 现代高性能并发编程：异步、多进程和多线程
- 神经网络和机器学习

## 本章摘要

本章介绍计算机科学的进阶主题,供有兴趣深入学习的读者参考。首先讲解现代高性能并发编程的核心概念,包括异步编程模型(async/await)、多进程和多线程的原理与实践、进程间通信(IPC)、线程同步机制(锁、信号量、条件变量)以及并发编程中的常见问题(死锁、竞态条件)和解决方案。接着介绍神经网络和机器学习的基础知识,包括神经网络的基本结构、反向传播算法、常见的机器学习框架(TensorFlow、PyTorch)以及深度学习的基本应用。这些内容为读者打开了通往高性能计算和人工智能领域的大门。
